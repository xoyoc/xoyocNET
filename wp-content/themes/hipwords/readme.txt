HipWords - Responsive Bootstrap WordPress Theme
===============================================

Clean and simple, lightweight and elegant blog. 
Responsive Bootstrap WordPress theme. 
Easy to use, very lightweight, clean design and fast loading. 
Made for developers, designers, bloggers and creative people.
